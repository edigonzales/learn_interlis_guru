= INTERLIS Lab Demo
:doctype: book

[.lead]
Dieses kleine Biblios-Demo zeigt fünf INTERLIS-Lernfragen mit der `interlis-lab-web-component` in einer klaren Progression von einfach zu anspruchsvoller.

[.exercise-meta]
Wichtig für dieses Demo:

- Die Biblios-Seite wird aus dem aktuellen Workspace gerendert.
- Die Web Component wird beim Build explizit in Version `0.1.11` aufgelöst.
- Jede Übung kann direkt im Browser kompiliert werden.

[.exercise-path]
== Lernpfad

1. Eine erste `CLASS` mit einem Attribut ergänzen
2. Obligatorische und optionale Attribute unterscheiden
3. Eigene `DOMAIN` und eine kleine Enumeration einführen
4. Compilerfehler bei Blöcken systematisch beheben
5. Eine erste `ASSOCIATION` zwischen zwei Klassen formulieren
